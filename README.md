## Overview

幫 Manzy 架了個 Fan club 來記錄 Manzy 參加了哪些活動，順便宣傳 Manzy 很可愛。因為這是我第一次用 Angular 寫網頁，所以可能寫的有點怪。

活動頁缺了很多照片，希望曼寶的粉絲們可以一起幫忙發 Issue 或 PR
